【js】问题汇总

haoge 2018-02-20 js汇总

# js问题汇总

  • 1.节点的种类有几种,分别是什么?

  • 1.元素节点：nodeType ===1

  • 2.文本节点：nodeType ===3

  • 3.属性节点:nodeType ===2

  • 2.列举javascript的3种主要数据类型,2种复合数据类型和2种特殊数据类型

  • 主要数据类型:string,boolean,number

  • 复合数据类型:function,object

  • 特殊类型:undefined,null

  • 3.new操作符具体是干什么的?

  • 1.创建一个空对象,并且this变量引用该对象,同时还继承了该函数的原型

  • 2.属性和方法被加入到this引用的对象中

  • 3.新创建的对象由this所引用,并且最后隐士的返回this

  • 4.null和undefined的区别?

  • 5.javascript的2种变量范围有什么不同？

  • 全局变量：当前页面内有效

  • 局部变量：函数方法内有效

  • 6.什么是函数的封装?

  • 把实现相同功能的代码放到一个函数体中,当想实现这个功能时,直接执行这个函数即可,减少了的 余;高内聚,低耦合

  • 7.Number的方法?

  • 1.isNaN: 强制转换成number,在判断

  • 2.Number:将其他数据类型的值强制转换成number类型

  • 3.paeseInt:经常用于字符串提取数字的方法

  • 4.parseFloat:和paeseInt用法一样,区别是多识别一位小数点

  • 5.toFixed：保留小数点位数的方法,返回值是一个字符串

  • 8.字符串的方法?

  • 1.toUpperCase:把小写字母转成大写

  • 2.toLowerCase:把大写转小写

  • 3.charAt:通过索引获取字符

  • 4.charCodeAt:通过索引获取对应字符的Unicode编码

  • 5.substr:截取substr(m,n)从索引m开始,截取n个字符

  • 6.substring:从索引m开始,截取到索引n,不包含n

  • 7.slice:从索引m开始,截取到索引n,不包含n

  • 8.indexOf:检测字符串中最后一次出现的索引位置

  • 9.LastIndexOf:检测字符在字符串中第一次出现的索引位置

  • 10.split:把字符串按照特定的字符分隔数组中的每一项

  • 11.replace:替换原有字符串不变,用新字符串替换旧的字符

  • 12.concat:拼接

  • 13.trim:去空格,去除字符串中左右的空格

  • 9.数组的方法?

  • 1.push:向数组末尾新增一项,可以传多个

  • 2.pop:删除数组的最后一项,不需要传参数

  • 3.unshift:向数组开头新增一项,不需要传参

  • 4.shift:删除数组的第一项,不需要传参

  • 5.sort:数组的排序

  • 6.reverse:将数组颠倒,原有数组发生改变

  • 7.concat:数组的拼接

  • 8.join:数组的连接

  • 9.indexOf:第一次出现的索引

  • 10.LastIndexOf:最后一次出现的索引位置

  • 11.map:数组的遍历和映射

  • 12.forEach:遍历数组

  • 13.toString:转字符串原有数组不发生改变

  • 14.slice:数组的截取

  • 10.Math的方法?

  • 1.Math.abs:取绝对值

  • 2.Math.floor:向下取整

  • 3.Math.ceil:向上取整

  • 4.Math.max:取最大值

  • 5.Math.min:取一组数的最小值

  • 6.Math.random:取随机数,取值范围(0,,1)

  • 7.Math.round:四舍五入取整

  • 8.Math.pow:取幂次方

  • 9.Math.sqrt:开平方

  • 11.获取元素的方法?

  • 1.document.getElementById:通过Id名来获取元素

  • 2.document.getElementByTagName:通过标签名来获取元素

  • 3.document.getElementByClassName:类数组集合

  • 4.docuemnt.getElementByName:通过name属性来获取元素

  • 5.document.documentElement:获取当前的html

  • Body:获取页面的body元素

  • document.querSelector:如果是id名加#,如果是class名加.

  • Document.querSelectorAll:获取所有的元素

  • 12.DOM节点的属性?

  • 1.childNodes:获取当前元素所有的子节点

  • 2.children:获取当前元素的子元素节点

  • 3.firstChild:获取第一子节点

  • 4.lastchild:获取最后一个子节点

  • 5.previousSIBling:获取上一个哥哥节点

  • 6.nextSibling:获取下一个弟弟节点

  • 7.parentNode：获取当前元素的父亲节点

  • 13.DOM的操作?

  • 1.创建新节点

  • createDocumentFragment:创建一个DOM片段

  • createElement:创建一个具体的元素

  • createTextNode:创建一个文本节点

  • 2.添加,移除,替换,插入

  • appendChild:添加

  • removeChild:移除

  • replaceChild:替换

  • insertBefore:插入(在已有的子节点前插入一个新的子节点)

  • 14.基本数据类型和引用数据类型两者存储的区别？

  • 基本数据类型存在栈区

  • 引用数据类型存在堆区和栈区

  • 15.递归的两个必要因素?

  • 递归方程

  • 递归结束条件

  • 16.改变this指向的方法?

  • call和apply

  • 17.js运行三步骤?

  • 1.全局扫描

  • 2.预编译/预解析

  • 3.解释执行

  • 18.函数预编译?

  • Ao(活跃对象)执行期上下文,运行期上下文

  • 1.创建Ao对象

  • 2.找形参和变量声明,将变量和形参作为Ao对象的属性名,值为undefined

  • 3.将实参形参相统一

  • 4.在函数体内里面找函数声明,值赋予函数体,在确定不被执行的代码块,函数声明只提升函数名(值undefined)

  • 19.全局预编译

  • 1.创建Go对象 Global Object

  • 2.变量声明,将变量作为Go对象的属性名,值为undefined

  • 3.在函数体里面找函数声明,值赋予函数体

  • 20.scope范围,什么是作用域?

  • [[scope]]:每个js函数都是一副对象,对象中有些属性可以访问,但有些就不可以,这些属性仅供js引擎存取,[[scope]]就是其中一- 个

  • 21.什么是作用域链?

  • [[scope]]中存储的就是执行期上下文的集合,这个集合承链式链接,我们把这个链式连接就叫做作用于链

  • 22.什么是闭包函数?

  • 当内部函数被保存到外部时,将会形成闭包

  • 闭包函数会导致原有的作用域链不释放,造成内存泄露

  • 23.闭包的作用?

  • 1.实现共有变量

  • 2.可以做缓存

  • 3.可以实现封装,属性私有化

  • 4.模块化开发,防止污染全局变量

  • 24.caller和callee的区别?

  • caller是返回调用该函数的引用

  • callee是返回本身自己的函数

  • 25.闭包的优缺点?

  • 优点:1.实现共有变量

  • 2.可以做缓存

  • 3.实现封装属性私有化

  • 4.模块儿化开发,防止污染全局变量

  • 缺点:内存泄漏

  • 26.构造函数在实例过程中(new)三段式?

  • 1.创建this = {};(隐士)

  • 2.this属性赋值

  • 3.返回this

  • 27.什么是原型?

  • 定义:原型是function对象的一个属性,它定义了构造函数制造出的对象的共有祖先,通过该构造函数产生的对象,可以继承该原型的属- 性和方法,原型也是对象

  • 28.原型,构造函数和对象三者的关系?

  • 1.函数找原型用prototype

  • 2.对象查看原型_proto

  • 3.对象查看原型constructor

  • 4.大多数对象都会继承自Object.prototype

  • 29.call和apply的区别?

  • 作用:改变this指向

  • 区别:传参形式不同

  • 30.this指向?

  • 普通函数this指向window

  • 构造函数this指实例出的对象

  • 全局this指向window

  • 31.DOM的全称?

  • DOM Document Object Model

  • DOM(宿主对象)定义了表示和修改文档所需的方法:是由浏览器产商定义,用来操作html和xml功能的一类对象的集合,也有称为html和- xml标准编程接口
